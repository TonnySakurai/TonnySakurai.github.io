Python学习记录之数据类型

Python提供了强大的对象类型这里的对象类型指的是c语言中数据结构实现的各种数据类型，如有序链表之类的作为语言的组成部分，在你利用Python解决问题之前往往没必要再重新编写对象的实现。事实上，除非你有内置类型无法提供的特殊的对象要处理，最好总是使用内置对象而不是使用自己的实现。

Python中没有类型声明。运行的表达式的语法决定了创建和使用的对象的类型。Python是动态类型的（它自动地追踪你的类型而不是要求声明代码），但它也是强类型语言（你只能对一个对象进行适合该类型的有效的操作）。

字符串

在python中，字符串作为序列存在。作为序列，字符串支持假设其中各个元素包含位置顺序的操作。包括以下操作：

获取字符串的长度

内置函数lan()可以获得字符串的长度：

>>> S = 'Spam'
>>> lan(S)
4

获取字符串其中某一个字符

S[a]代表相对于第一个字符偏移a位的字符，即:

>>> S[1]
'p'

反向索引

Python支持反向索引，即:

>>> S[-1]
'm'

分片

Python支持分片(slice)的操作，这是一种一步就能够提取整个分片的方法，即:

>>> S[1:3]
'pa'

分片的一般形式为X[I:J]，表示“取出在X中从偏移量为I，直到但不包括偏移量为J的内容”。在一个分片中，左边界默认为0，并且右边界默认为分片序列的长度，如以下例子所示：

>>> S[ :3] 
'Spa'    
>>> S[ :-1] 
'Spa'    
>>> S[ : ]
'Spam'

字符串的合并

作为一个序列，字符串也支持使用加号进行合并（将两个字符串合并成为一个新的字符串），或者重复。例如：

>>> S + 'xyz'
'Spamxyz'    
>>> S*2  
'SpamSpam'

字符串具有不可变性

字符串在python中具有不可变性——在创建之后不能就地改变。例如不能通过对其某一位置进行赋值而改变字符串，如下操作是非法的：

>>> S[0] = 'Z'

但可以通过建立一个新的字符串并以同一个变量名对其进行赋值以达成目的：

>>> S = 'Z' + S[1: ]
>>> S
'Zpam'

列表

序列操作：

由于列表是序列的一种，列表支持所有的我们对于字符串讨论过的序列操作，如：

>>> L = [123, 'spam', 1.23]
>>> len(L) 
3    
>>> L[0]
123    
>>> L[ :-1] 
[123, 'spam']    
>>> L + [4, 5, 6] 
[123, 'spam', 1.23, 4, 5, 6]

类型特定的操作：

增加、减少列表的大小：

>>> L.append('NI')    #增加值为NI的一项
>>> L
[123, 'spam', 1.23, 'NI']

>>> L.pop(2)          #移除给定偏移量为2的一项
1.23
>>> L
[123, 'spam', 'NI']

边界检查：

超出列表末尾之外的索引或赋值会导致错误。

嵌套：

Python核心数据类型的一个优秀的特性就是它们支持任意的嵌套。能够以任意的组合对其进行嵌套，并可以多个层次进行嵌套（例如，能够让一个列表包含一个字典，并在这个字典中包含另一个列表等）。这种特性的一个直接的应用就是实现矩阵。

>>> M = [[1, 2, 3],
          [4, 5, 6],
          [7, 8, 9]]
>>> M[1]
[4, 5, 6]
>>> M[1][2]
6

注：这种矩阵结构适用于小规模的任务，但对于更重要的数值运算而言，可以考虑使用Python数值扩展包，如NumPy系统。

列表解析（list comprehension expression）：

列表解析源自集合的概念。它是一种通过对序列中的每一项运行一个表达式的方式来创建一个新的列表的方法。如下面例子所示：

>>> col2 = [row[1] for row in M]       #把矩阵M的每个row中的row[1]，放在一个新的列表中
>>> col2 
[2, 5, 8]
>>> [row[1] + 1 for row in M]
[3, 6, 9]
>>> [row[1] for row in M if row[1] % 2 == 0]
[2, 8]
>>> diag = [M[i][i] for i in [0, 1, 2] ]
[1, 5, 9]
>>> doubles = [c * 2 for c in 'spam']
>>> doubles
['ss', 'pp', 'aa', 'mm']
